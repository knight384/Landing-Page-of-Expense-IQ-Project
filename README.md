# ExpenseIQ - Precision Minimalist Expense Tracking

ExpenseIQ is a polished, production-ready web application designed for modern savers who value financial transparency and data-driven insights. Built with a focus on "Intelligence through Clarity," it provides a sophisticated interface for tracking expenses, setting budgets, and visualizing financial health.

## 🚀 Features

- **Smart Expense Tracking**: Quickly log and categorize transactions as they happen.
- **Intuitive Budgeting**: Define spending limits for different categories to stay on track.
- **Advanced Data Visualization**: Interactive charts and peaking indicators for financial trends.
- **Mobile-First Design**: Fully responsive interface that works seamlessly on any device.
- **Interactive Top Navigation**: Smooth scrolling and active section highlighting for better UX.
- **Modern Tech Stack**: Built with React 19, Tailwind CSS 4, and Framer Motion for fluid animations.

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS 4.0
- **Animations**: Motion (framer-motion)
- **Icons**: Lucide React
- **Typography**: Manrope (Headings) & Inter (Body)
- **Development**: TypeScript

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) (v9 or higher)

## ⚙️ Setup Instructions

### 1. Clone the Repository
```bash
git clone <your-repository-url>
cd expenseiq
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Environment Configuration
Create a `.env` file in the root directory based on `.env.example`:
```env
GEMINI_API_KEY="your_api_key_here"
```
*Note: Ensure you have a valid Gemini API key from Google AI Studio if using AI features.*

### 4. Start Development Server
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### 5. Build for Production
```bash
npm run build
```
Static files will be generated in the `dist/` directory.

## 🎨 Design Philosophy

ExpenseIQ follows the **Precision Minimalist** design system:
- **Clarity over Clutter**: Generous whitespace and high-quality typography.
- **Dual-Font Strategy**: Manrope for refined headlines; Inter for utilitarian body text.
- **Sophisticated Elevation**: Soft ambient shadows and tonal layering rather than flat surfaces.
- **Approachable Shapes**: Rounded corners (8px-24px) to balance financial rigor with high usability.
